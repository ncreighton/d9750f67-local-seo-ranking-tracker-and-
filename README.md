== Description ==

A WordPress plugin for local businesses, marketing agencies, and multi-location brands tracking and improving Google local search rankings. Monitors keyword positions in the local pack for target cities, audits Google Business Profile completeness and citation consistency, identifies NAP discrepancies across directories, sends weekly ranking movement reports, and provides actionable local SEO recommendations inside the WordPress admin dashboard.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Local SEO Ranking Tracker and Google Business Profile Optimizer WordPress Plugin
